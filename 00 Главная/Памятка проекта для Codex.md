# Памятка проекта для Codex

## Задачи и рекомендации
- Рекомендация из врачебного документа не становится задачей автоматически, если это не отдельное понятное действие для семьи.
- Если несколько рекомендаций относятся к одному реальному действию, их нужно объединять в одну сборную задачу.
- Сборная задача хранится как `kind: grouped_task`, получает `task_page: true`, список `items` и все связанные `grouped_source_event_ids`.
- Карточка сборной задачи на сайте должна вести на отдельную страницу `/tasks/<slug>`, а не на одно случайное событие-источник.
- На странице сборной задачи нужно показывать, что сделать, из каких событий/документов это взято, и давать ссылки на каждое исходное медицинское событие.
- Обычные одиночные задачи могут вести прямо на связанное событие.
