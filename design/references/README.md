# Design References

This folder stores approved visual references for "База Бабиновых" and its **Dark Family OS** direction.

Images in `design/references/` are the primary visual reference for:

- dark theme mood;
- card density;
- circular avatars;
- five-color semantic palette;
- person profile structure;
- metrics drill-down;
- tasks drill-down;
- appointments drill-down;
- mobile adaptation.

## How To Use References

Do not copy pixel-perfect. Match:

- mood;
- hierarchy;
- spacing;
- card softness;
- entity structure;
- right detail drawers;
- mobile behavior.

Use references to judge whether the app feels like a premium private family OS rather than a generic admin dashboard.

## Suggested Usage Order

1. `01-dashboard-dark.png` - global dashboard mood and app shell.
2. `02-profile-summary.png` - person profile summary and "Что важно сейчас".
3. `03-profile-deep.png` - deeper person page structure.
4. `04-metrics-expanded.png` - metrics page and indicator drill-down.
5. `05-appointments-expanded.png` - appointments page and appointment detail.
6. `06-tasks-expanded.png` - tasks page and task detail.

## What To Add Here

Useful files include:

- screenshots of current app screens;
- screenshots of target mood and density;
- annotated references for profile pages, document indexes, task systems, indicators, appointments, and mobile views;
- notes explaining what to borrow and what to avoid.

## Privacy

Do not add sensitive medical information, personal identifiers, private family records, lab results, prescriptions, or source documents to exported visual references unless the user explicitly approves it.
