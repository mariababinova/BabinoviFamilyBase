# DESIGN.md — База Бабиновых

## Product identity

“База Бабиновых” is a private family operating system.

It is not an admin dashboard.
It is not a medical-only app.
It is not a generic SaaS interface.

It is a calm, premium, dark family portal where life is organized into clear shelves.

Main phrase:

`Жизнь разложена по полочкам`

## Top-level modules

- Главная
- Здоровье
- Быт
- Места
- Дела
- Документы

## Visual direction

Dark, airy, glassy, soft, elegant.

The interface should feel:
- private;
- expensive;
- calm;
- family-oriented;
- useful every day;
- structured but not bureaucratic.

## What to avoid

Avoid:
- admin dashboard look;
- dense data blocks;
- too many counters;
- tables on portal pages;
- generic cards;
- harsh contrast;
- random colors;
- clutter;
- over-bold typography.

## Home page

Home page is a portal.

It should include:
- brand title;
- tagline;
- module cards;
- quick actions.

It should not include:
- health metrics;
- people cards;
- documents list;
- event feed;
- task list;
- “Сегодня важно”, unless explicitly requested.

## Main visual reference

The current accepted home page direction:
- large elegant title;
- dark glass background;
- left sidebar with module icons;
- five module cards with large circular glowing icons;
- quick action cards;
- no family member list;
- no bottom user block;
- no “Сегодня важно”.

## Typography

Preferred:
- Onest;
- Manrope;
- Inter;
- SF Pro;
- system-ui.

Hero titles should be lighter and elegant.
Avoid huge bold 900-weight titles on portal pages.

## Icons

Icons are a major part of the style.

Use line icons with a consistent stroke.

Portal module icons:
- Здоровье: heart / pulse
- Быт: home
- Места: map pin
- Дела: check-square
- Документы: folder

Action icons:
- Загрузить документ: upload / file plus
- Добавить задачу: plus / check
- Добавить место: map pin
- Найти в базе: search

## Page philosophy

Each page has one job.

Главная:
choose module.

Здоровье:
choose health profile or health action.

Person profile:
show detailed health data.

Места:
explore saved places.

Дела:
manage delegated tasks.

Документы:
find family documents.
