# Visual Review Checklist

Before accepting UI changes, review:

- Is the page focused on one job?
- Is it too dense?
- Does it look like a generic admin dashboard?
- Are cards soft, airy, and spacious?
- Are icons meaningful and consistent?
- Is typography modern and not too heavy?
- Are colors semantic?
- Are there random counters that do not help the user?
- Is there unnecessary duplicated navigation?
- Does the page feel like a family OS?
- Did the implementation avoid unrelated pages?
- Did it preserve data and routes?
