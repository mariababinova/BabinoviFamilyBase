# Дела

Этот раздел хранит ассистентский задачник семьи отдельно от медицинских контрольных задач.

## Файлы

- `assistant_tasks.json` - импортированные задачи из Trello для страницы `/affairs`.

## Импорт из Trello

Команда запускается из папки `06 Сайт`:

```powershell
npm run import:trello:affairs
```

Нужны переменные в `.env` корня репозитория:

```env
TRELLO_API_KEY=...
TRELLO_TOKEN=...
TRELLO_BOARD_ID=...
```

`TRELLO_BOARD_ID` необязателен: если он не указан, импортёр ищет доску `Задачи`.

Импорт read-only по отношению к Trello: он только читает карточки, списки, labels, сроки и описания. В Trello ничего не меняется.
